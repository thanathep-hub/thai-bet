## Design Context

### Users
เจ้าของร้านขายหวย (ร้านเจ๊หน่อย) — ใช้คนเดียว ใช้บนมือถือขณะนั่งอยู่ที่ร้านหรือรับออเดอร์จากลูกค้า งานหลักคือบันทึกการลงหวย ดูประวัติ และเช็คผล ต้องการความเร็วและความถูกต้อง ไม่ใช่ความสวยงาม

### Brand Personality
- **3 คำ**: เร็ว · ชัดเจน · เชื่อถือได้
- **Tone**: เครื่องมือทำงาน ไม่ใช่แอปผู้บริโภค — เหมือน POS หรือแอปธนาคาร ไม่ใช่ e-commerce
- **Emotional goal**: รู้สึกควบคุมได้ มีข้อมูลครบ ทำงานได้เร็วโดยไม่ผิดพลาด

### Aesthetic Direction
- **Visual tone**: Clean utility — สะอาด มีโครงสร้างชัด ไม่มีลวดลายเกินความจำเป็น
- **Primary color**: เขียว (#0a7d55) — สื่อถึงเงิน ความมั่นคง
- **Typography**: Sarabun — อ่านง่ายทั้ง Thai และตัวเลข
- **Theme**: Light mode หลัก มี dark mode สำรับ
- **Anti-references**: ไม่ใช่แอปเกม ไม่ใช่แอปความบันเทิง ไม่ใช่ landing page
- **Most critical page**: index.html (รายชื่อลูกค้า) — ต้องสแกนและเข้าถึงได้เร็วที่สุด

### Design Principles
1. **Speed over beauty** — ทุก interaction ต้องเร็ว tap น้อยครั้งที่สุด ไม่มี animation ที่บล็อก task
2. **Data clarity first** — ตัวเลขและสถานะต้องเห็นชัดก่อนทุกสิ่ง ใช้ contrast และ size ไม่ใช่ decoration
3. **Thumb-zone design** — action หลักต้องอยู่ในโซนนิ้วหัวแม่มือบน mobile; บน desktop ใช้ sidebar แทน bottom nav
4. **No cognitive load** — layout ซ้ำกันทุกหน้า เรียนรู้ครั้งเดียวใช้ได้ทุกที่ — หน้าตาต่างกันตาม screen แต่ mental model เดิม
5. **Adapt, never amputate** — ทุก feature ต้องมีอยู่ทุก screen ขนาด ไม่มีการซ่อนหรือตัดออก layout เปลี่ยนแต่ functionality ครบ

### Responsive Breakpoints
| Breakpoint | Range | Layout |
|-----------|-------|--------|
| Mobile | 320–767px | Single column, bottom nav, thumb-first |
| Tablet | 768–1023px | 2-column card grid, bottom nav ยังอยู่ |
| Desktop | 1024px+ | Sidebar nav (always visible) + main content, multi-column |

### Cross-screen Rules
- **Navigation**: bottom nav → sidebar nav ที่ 1024px (ไม่ใช่ hamburger)
- **Card grid**: 1 col → 2 col ที่ 768px — ไม่เกิน 3 col (ข้อมูลอ่านยาก)
- **Touch targets**: 44px min ทุก size — บน desktop ยิ่งง่ายกว่า
- **Typography**: ขนาดเท่าเดิม — ไม่ต้องขยาย เพราะ desktop ดูจากระยะไกลกว่า
- **Hero card**: full-width ทุก size, stat ขยาย horizontal บน desktop
